# mips_combiler_16_bit_FEM
A functional 16 bit MIPS CPU simulator. <br>

## - Instruction Types
With this project you can run the assembly commands below

####   R Types
`add, and, sub, or, xor, jr, sll, srl`

####   I Types
`addi, andi, subi, ori, beq, bne, lw, sw`

####   J Types
`j, jal`
